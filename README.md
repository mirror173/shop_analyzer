# 店铺业绩分析系统

一个基于Streamlit的店铺业绩分析工具，可以上传Excel文件进行数据分析。

## 功能特点

- 📊 数据概览仪表板
- 📦 产品表现分析
- 📈 月度对比分析
- 📁 品类分析
- 🚚 运费分析
- 📏 尺寸分析
- 📈 趋势分析
- 💾 数据导出功能

## 使用说明

1. 在左侧边栏上传Excel文件
2. 选择分析类型
3. 查看分析结果和图表
4. 下载分析报告

## 数据格式要求

Excel文件应包含以下列：
- 订单编号、商品中文名称、商品数量
- 商品销售单价、商品总金额
- 运费收入、日期、SKU
- 品类相关列（可选）

## 在线应用

点击右上角"App"按钮开始使用
